# FishExam

Diese App soll dabei helfen, euch auf die Fischreiprüfung vorzubereiten.

Es gibt einen Lernbereich für die Fragen, sowie für die Fische. Ebenso gibt es einen Prüfungsbreich für die Fragen sowie für die Fische.

Die Prüfungen sind nach den offiziellen Richtlinien.


> [!NOTE]  
> Die Fragen, Fische sowie die Prüfungsrichtlinien stammen aus dem Land NRW.
> Inwiefern sich diese Informationen zu anderen Bundesländern unterscheiden, kann ich nicht sagen.

Für Verbesserungsvorschäge bin ich jederzeit offen.
